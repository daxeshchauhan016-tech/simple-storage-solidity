# Simple Storage Smart Contract

## Description
A basic Solidity smart contract that stores and retrieves a number.
It also tracks the last updater and emits events.

## Functions
- setNumber(uint256)
- getNumber()

## Network
- Sepolia Testnet

## Deployed Address
0xYOUR_CONTRACT_ADDRESS

## How to Test
1. Open Remix
2. Paste contract
3. Deploy on Sepolia
4. Call setNumber & getNumber
